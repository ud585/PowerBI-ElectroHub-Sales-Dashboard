# 📊PowerBI-ElectroHub-Sales-Dashboard

## 📌 Project Overview

This Power BI project analyzes the sales performance of **ElectroHub**, a retail company dealing in Electronics, Clothing, Footwear, Home Appliances, Kitchenware, Accessories, Bags, and Personal Care products.

The dashboard provides interactive business insights using Power BI visualizations, DAX measures, and data modeling techniques to help stakeholders make data-driven decisions.

---

# 🎯 Business Requirements

The dashboard satisfies the following business requirements:

* ✅ Top 5 & Bottom 5 Products by Sales/Profit/Quantity Sold
* ✅ Sales Trend Analysis (Daily, Monthly, Quarterly & Yearly)
* ✅ Sales vs Profit Relationship
* ✅ Compare Sales, Profit & Quantity between two selected periods
* ✅ Average Discount by Discount Category
* ✅ Total Number of Orders
* ✅ Detailed Order Table with Filters
* ✅ Sales by City
